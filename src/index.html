Seed Viability using X-Ray Imaging

In collaboration with Kew Gardens, we developed an interactive web-app in React that used a Flask API to process and classify images of seeds using fine-tuned implementations of YOLOv8 and Mask R-CNN.

The product was used by Kew Gardens to classify seeds and provide growers with information on seed viability, massively improving the efficiency of seed classification.

Discovering Diverse Artificial Life using Quality Diversity

For my Master's Thesis, I explored diverse Artificial Life using Quality Diversity in Lenia. I enhanced the QD algorithm AURORA to learn rotationally invariant descriptors.

AURORA used dimensionality reduction via a variational autoencoder to generate evolutionary algorithm descriptors. I implemented contrastive learning methods like NT-Xent and Triplet-Margin learning, and Group Theoretic approaches for rotationally invariant embeddings.

I demonstrated that extinction events could drive the discovery of new behaviours and creatures in Lenia, paving the way for open-ended creature discovery, a result I aim to publish!
